# Privacy Policy – IMVU Promoter Hub

_Last updated: February 2026_

## 1. Overview

IMVU Promoter Hub is a browser extension that assists IMVU creators with routine on-platform actions. This Privacy Policy explains what information the extension accesses, what it does not collect, and how anonymous analytics data is handled.

## 2. Data Collection

IMVU Promoter Hub does **not** collect, store, or transmit any of the following:

- Passwords or login credentials
- Private messages or conversation content
- Payment or billing information
- Authentication tokens
- Browser cookies

The extension operates exclusively on IMVU web pages and does not access content on any other website.

## 3. Analytics

Anonymous usage analytics may be collected to help improve the performance and stability of the extension. This data is strictly non-personal and may include information such as:

- Extension version number
- Feature usage events (e.g., which features are activated)
- Process statistics (e.g., task completion counts)
- Error events and diagnostic information

No personally identifiable information (PII) is included in or linked to any analytics data.

## 4. Data Sharing

We do not sell, rent, or share any user data with third parties. Analytics data, where collected, is used solely for internal product improvement purposes.

## 5. Permissions Usage

IMVU Promoter Hub requests the following browser permissions, each used only for its stated purpose:

- **activeTab** – Allows the extension to interact with the currently active IMVU tab when the user initiates an action.
- **scripting** – Enables the extension to run helper scripts on IMVU pages to perform on-platform actions on behalf of the user.
- **storage** – Used to save user preferences and settings locally in the browser.
- **notifications** – Used to inform the user about the status of ongoing tasks or important updates.

IMVU Promoter Hub is an independent browser extension and is not affiliated with or endorsed by IMVU.
